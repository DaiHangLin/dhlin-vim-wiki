记录vim中常用的几个操作

入门指南

    $ vimtutor

1. vim中是区分大小写
2. vim中移动光标

        h  向左移动        
        j  向下移动        
        k  向上移动        
        l  向右移动

    其实使用方向键也是能移动的，但是熟悉后再一些其他的编辑器或者浏览器中也能使用

3. vim 中删除

        x 删除光标所在的字母
        dw 删除完整的单词不包括后面的标点
        dd 删除整行
        D  删除整行
 

    ## 2017-4-25

4. vim中移动行 

        :m 12     如果当前行数为第10行，则将当前行移动到第12行
        如果当前行数为第20行，则将当前行移动到第13行
        
        :m 0       将当前行移动到第一行前面
        
        :m $       将当前行移动到最后一行后面
        :m0        后面的空格可以省略不写

5. vim中移动多行

        :m5,7m 21 将第5，6，7行移动到第21行
        
        :m5,7m 0 将第5，6，7行移动到第一行前面
        
        :m5,7m $ 将第5，6，7行移动到最后一行后面
        
        :.,.+4m 21 将从当前行开始的5行，移动到21行位置
        
        :,+4m21 和上个命令一样，可以省略.和空格

6. vim中撤销上一个操作

        u 撤销
